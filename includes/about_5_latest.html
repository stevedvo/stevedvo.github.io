06 September 2016

> Spent another few hours working through my "WordPress In Easy Steps" book, and actually managed to get to the end of it already! The chapters covered in the last half of the book are:

  • Settings & Tools
  • SEO & Social Media
    • WordPress SEO by Yoast (plugin)
    • Social Media Buttons
    • Twitter Feed
    • Instagram Photos
  • User Interaction
    • Comments
    • Spam Control
    • Forums
    • Contact Forms
    • Translation
    • Membership Levels
  • Tips & Tricks
    • Google Analytics
    • Google Maps
    • Newsletter Forms
    • Sitemaps
    • Google Fonts
  • Advanced WordPress
    • WordPress & PHP
    • Setting Up FTP
    • Creating A Theme
    • WordPress Backup
    • Responsive Design

> Almost all of the chapters basically involved downloading some sort of plugin, maybe signing up to some sort of service to get an API key, and covering how to configure various settings with each chosen plugin - most of which were fairly obvious in any case. Whilst it was good to have this available as a reference, I have to say this left me feeling rather disappointed. There was only really the last chapter that seemed to have some good meaty content about how to interact with the backend but it was only briefly touched upon and doesn't go into great depth/detail.

> I got fed up of trying to do anything on my live 123-Reg server since it was mind-numbingly slow, so I did what I could on my local WAMP server - which was a bit limiting in terms of the services that I could sign up with and get API keys for.

> I can see that the book IS a very useful guide/resource for the layperson wanting to understand the basics of WordPress to set up their own website, and is good for pointing out particular good plugins for achieving various tasks depending on what you might want to use the website for... but I was hoping for more technical stuff to understand how to interact with WordPress and do more theme creation / tweaking - perhaps even creating my own plugins. I expect there are other books / online resources available that will help me get up to this level.
